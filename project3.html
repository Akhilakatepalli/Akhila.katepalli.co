Project Overview

-Designed and implemented a biometric-based authentication protocol for secure cloud access. -Developed the solution using Java and managed data with a robust database. -Created intuitive and responsive user interfaces using HTML and CSS to enhance user experience. -Utilized biometric data to create unique user identities and generate private keys, eliminating the need for key storage and prior information sharing. -Efficiently generated session keys using biometric templates to ensure secure communication. -Rigorously assessed the protocol’s security with AVISPA, confirming its resilience against known attacks. Demonstrated the protocol’s effectiveness and practicality through extensive testing.

Technologies Used:

  • HTML, CSS, JavaScript
  • Node.js
  • MySQL
  • GitHub for version control
