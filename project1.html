Project Overview

-Developed HotelWise, a hotel management system to streamline hospitality operations. -Utilized Python Flask for backend development, HTML/CSS for frontend, and MongoDB for data storage. -Provided a centralized admin dashboard for managing amenities, room availability, reservations, and billing. -Enabled hotel officers to manage updates and guest interactions with restricted access. Allowed guests to self-register, browse amenities, make bookings, and manage bills online. -Enhanced operational efficiency and guest experiences through effective data management and intuitive interfaces.

Technologies Used:

  • HTML, CSS, JavaScript
  • Node.js
  • MySQL
  • GitHub for version control
